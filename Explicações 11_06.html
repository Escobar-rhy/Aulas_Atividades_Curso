Slice e Splice

Exemplo de Slice: Para extrair e não modificar o array original (cria uma "fatia").

const frutas = ['maçã', 'banana', 'laranja', 'uva', 'kiwi'];
// Extrai do índice 1 (banana) até o índice 4 (uva - não incluído)
const algumasFrutas = frutas.slice(1, 4);
console.log(algumasFrutas); // Saída: ['banana', 'laranja', 'uva']
console.log(frutas); // Saída: ['maçã', 'banana', 'laranja', 'uva', 'kiwi'] (original intacto)

Exemplo de Splice: Para modificar o array original (adicionar, remover ou substituir elementos).

// ex de remoção
const cores = ['vermelho', 'verde', 'azul', 'amarelo'];
// A partir do índice 1, remove 2 elementos
const coresRemovidas = cores.splice(1, 2);
console.log(cores); // Saída: ['vermelho', 'amarelo'] (array original modificado)
console.log(coresRemovidas); // Saída: ['verde', 'azul'] (elementos que foram removidos)

// ex de adição
const numeros = [1, 2, 5, 6];
// A partir do índice 2, remove 0 elementos, e adiciona 3 e 4
numeros.splice(2, 0, 3, 4);
console.log(numeros); // Saída: [1, 2, 3, 4, 5, 6]

// ex de substituição
const letras = ['a', 'b', 'c', 'd'];
// A partir do índice 1, remove 1 elemento ('b') e adiciona 'x' e 'y'
letras.splice(1, 1, 'x', 'y');
console.log(letras); // Saída: ['a', 'x', 'y', 'c', 'd']

ForEach: é um método de array em JavaScript que serve para executar uma função uma vez para cada elemento em um array.

// ex foreach
const nomes = ['Ana', 'Bruno', 'Carla', 'Diego'];
// Usando forEach para saudar cada nome
nomes.forEach(function(nomeAtual, indice) {
console.log(`Olá, ${nomeAtual}! (Índice: ${indice})`);
}); /* Saída no console:
Olá, Ana! (Índice: 0)
Olá, Bruno! (Índice: 1)
Olá, Carla! (Índice: 2)
Olá, Diego! (Índice: 3) */

// ex 2 foreach
const produtos = ['Laptop', 'Mouse', 'Teclado', 'Monitor'];
// Imprime cada produto em maiúsculas
produtos.forEach(produto => {
console.log(produto.toUpperCase());
}); /* Saída no console:
LAPTOP
MOUSE
TECLADO
MONITOR */

Arrow Function: são uma maneira mais concisa de escrever funções em JavaScript. O nome "arrow" (seta) vem do operador => que é usado na sua declaração.
